# Redmatch 2 - RU Localization Fix

> Неофициальный проект, основанный на личном видении автора. Не связан с
> разработчиками игры и не является официальным.

Репозиторий содержит исправления русского перевода Redmatch 2 и скрипт, который
применяет их к файлу игры `localization.csv`.

## Быстрый старт

```powershell
py script.py "C:\path\to\localization.csv"
```

Скрипт перезапишет `localization.csv` и создаст рядом резервную копию:
`localization.bak_YYYYMMDD_HHMMSS.csv`.

## Состав репозитория

- `LocalizationRemake.csv`: источник исправленного RU-перевода.
- `script.py`: обновляет колонку `russian` в `localization.csv`.

## О проекте

Русский перевод Redmatch 2 местами неполон и, похоже, давно не обновлялся: в нём
встречаются неточности, разный стиль терминов и строки, оставшиеся на английском
(например, оружие `Double Barrel`).

В рамках проекта подготовлена альтернативная версия русского перевода, основанная
на оригинальном английском тексте и стремлении привести локализацию к более
цельному и актуальному виду.

Проект носит некоммерческий характер и создан для личного использования, при этом
может быть полезен и другим.

## Что исправлено

Правки делают RU-перевод точнее и единообразнее:

- Обновлены кредиты переводчиков.
- Упрощены названия меню и карт: "карты", "стандартные", "пользовательские".
- Приведены к одному стилю термины: ресурс-пак, мастерская Steam.
- "комната" заменена на "матч".
- Переписаны сообщения про античит и режим наблюдателя.
- Исправлены опечатки, регистр и пунктуация.
- Добавлены недостающие строки: модерация, невидимость, двустволка.
- Более удачно переведены достижения и термины оружия.

## Требования

- Python 3.7 или новее
- Установленная Redmatch 2 (нужен файл `localization.csv`)

## Установка и использование

1. Скачайте репозиторий (или распакуйте архив) в отдельную папку.
2. Убедитесь, что рядом лежат `script.py` и `LocalizationRemake.csv`.
3. Закройте игру, если она запущена (файл может быть занят).
4. Запустите скрипт, передав путь к `localization.csv`:

   ```powershell
   py script.py "C:\path\to\localization.csv"
   ```

   Если команда `py` не работает, используйте:

   ```powershell
   python script.py "C:\path\to\localization.csv"
   ```

5. Файл обычно находится по пути:

   ```text
   C:\Program Files (x86)\Steam\steamapps\common\Redmatch 2\Redmatch 2_Data\StreamingAssets\localization.csv
   ```

6. Если игра установлена в другом месте, найдите файл
   `Redmatch 2_Data\StreamingAssets\localization.csv`.

   Быстрый способ открыть папку игры: Steam > Библиотека > ПКМ по Redmatch 2 >
   Управление > Просмотреть локальные файлы.

Скрипт обновляет только колонку `russian`, используя данные из
`LocalizationRemake.csv`.

> **Важно:** скрипт перезаписывает `localization.csv`. Рядом создаётся резервная
> копия: `localization.bak_YYYYMMDD_HHMMSS.csv`.

Пример вывода:

```text
Done.
Backup created: C:\path\to\localization.bak_YYYYMMDD_HHMMSS.csv
Russian rows updated: 123
Keys missing in remake: 0
```

## Как это работает

- Строки сопоставляются по колонке `Text Key (internal use only)`.
- Изменяется только колонка `russian`; остальные данные не трогаются.
- Перед заменой создаётся резервная копия, затем файл перезаписывается через
  временный файл с расширением `.tmp`.
- Чтение/запись выполняются в кодировке UTF-8 (с BOM).

## Откат

Чтобы вернуть исходный перевод:

1. Найдите файл резервной копии `localization.bak_*.csv`.
2. Переименуйте текущий `localization.csv` (например, в `localization.csv.broken`).
3. Переименуйте резервную копию в `localization.csv`.

## Возможные проблемы

- `Target file not found`: неверный путь к `localization.csv`.
- `Required column not found`: файл не того формата/версии (нужны колонки
  `Text Key (internal use only)` и `russian`).
- `Permission denied` / `Access is denied`: закройте игру и убедитесь, что файл
  не открыт и доступен для записи.
